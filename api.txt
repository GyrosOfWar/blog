Base URL: /api
    GET /user
        Retreives user details for logged-in user. Authenticated

    GET /user/<user_id:int>/post/<post_id:int>
        Gets post with ID for user

    GET /user/<id:int>/posts?start=int&limit=int
        Get a page of posts by the specified user. Not authenticated 
    
    GET /user/<id:int>/posts?tag=string&start=int&limit=int 
        Get a page of posts with specified tag 

    GET /search?q=string
        Search posts with the query
